## EcoSort
EcoSort is a modern, AI-inspired sustainability web app that helps users understand how to properly sort waste, learn local disposal rules, and stay motivated through a gamified rewards system.

## Features

- **Smart Waste Classification**  
  Instantly identify how to dispose of common waste items using intuitive UI flows.

- **Global Disposal Rules**  
  Explore general waste-sorting guidelines that apply across regions.

- **Gamified Rewards System**  
  Earn points, unlock rewards, and track progress to encourage sustainable habits.

- **Educational-First Design**  
  Clear explanations, friendly copy, and visual cues make learning easy and engaging.

- **Modern Eco-Friendly UI**  
  Clean layouts, soft gradients, and nature-inspired visuals for a calm user experience.
